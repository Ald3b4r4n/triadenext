# Design System

## Identidade

Tríade Essenza Parfum usa uma estética premium, quente e editorial: fundos creme, tipografia serifada para desejo/produto, vinho escuro para ação e champagne como acento de luxo.

## Componentes base

- Header público.
- Navegação de categoria.
- Hero editorial.
- Card de produto.
- CTA primário e secundário.
- Badge de status.
- Form field.
- Card administrativo.
- Tabela administrativa responsiva.
- Toast/alerta.

## Diferenças intencionais

- O alvo Next pode modernizar layout e interações.
- A estética deve preservar sensação de perfumaria premium.
- Dependências visuais específicas do legado não são obrigatórias.

## Critérios de aceite

- Nenhuma tela pública deve parecer placeholder.
- Mobile deve ser primeira classe.
- Estados vazios e erros devem ser explícitos e seguros.
- Componentes administrativos devem priorizar clareza operacional.
