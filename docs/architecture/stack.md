# Stack

Stack alvo:

- Next.js App Router com TypeScript.
- Vercel para hosting.
- Neon Postgres com Drizzle ORM.
- Vercel Blob para imagens e documentos fiscais.
- Stripe para checkout e webhooks em fase posterior.
- Tailwind CSS, shadcn/ui, Radix UI, Zod e React Hook Form.
- Vitest, Testing Library e Playwright.
- ESLint e Prettier.
