# Requirements - Dashboard Admin Operacional

## Objetivo

Substituir dashboard administrativo mínimo por visão operacional útil para loja, pedidos, alertas e ações pendentes.

## Escopo

- Cards de KPIs operacionais.
- Pedidos recentes e pendências.
- Alertas de estoque, pagamento, frete e fiscal.
- Links para ações administrativas existentes.
- Estados vazios e erros seguros.

## Regras

- Admin exige sessão e papel autorizado.
- Métricas devem usar agregações seguras.
- Não expor dados pessoais desnecessários.

## Fora do escopo

- BI avançado.
- Exportações fiscais/financeiras.
